How to use these decks

  1. Open the deck for the week you are teaching in a Chromium-based browser (Edge, Chrome).
  2. Press F to go fullscreen.
  3. In Microsoft Teams, share the browser window (not the desktop).
  4. Use → / Space to advance, ← to go back, and N to view your speaker notes off-screen.
  5. Number keys jump to a slide (e.g. 25Enter).

Download as PDF

Print → Save as PDF

Each deck includes a print stylesheet that lays out one slide per page followed by its speaker notes — suitable for offline review or printed handouts. To export:

  1. Open the deck in Chrome or Edge.
  2. Press P from inside the deck (or use the browser’s File → Print menu / Ctrl+P).
  3. In the print dialog, set Destination to “Save as PDF.”
  4. Set Layout to Landscape, Margins to None, and enable Background graphics.
  5. Click Save.

The browser’s default print preview shows one slide per page with its speaker notes following on the next page. There is no separate PDF file to download — the deck’s print stylesheet generates the handout on demand.

Visual identity

The deck system — palette, type scale, layouts, navigation contract — is documented in DESIGN.md. All six weekly decks share a single css/deck.css and js/deck.js; do not fork them.
