Student Handouts

One-page printable take-homes for each weekly session. Each handout lists what to bring, the key terms you’ll hear, the exercises you’ll do live, what you should be able to do by the end, and the homework that gates next week — all on a single landscape page you can print or save before class.

How to use a handout

  1. Print the handout for your week before the session, or save it as a PDF on your laptop.
  2. Print landscape, single-sided, A4 or US Letter. Use Chrome or Edge for best fidelity (Ctrl+P → Layout Landscape → Margins Default → Background graphics on).
  3. Bring it to the session. Mark up the exercises as you work, and circle the homework before you log off.
  4. The handout is the student companion to the slide deck. Instructors carry the matching Facilitator Pack.

What each handout contains

  • Header strip — course name, week, duration, audience, prerequisites at a glance.
  • Module strip — every module with start time and posture (talk / build / break) so you know where you are.
  • Bring with you — accounts, tools, reading, and the artefacts (notes, prototype, frontier map) you owe before walking in.
  • Key terms — the vocabulary you’ll hear in the room, defined in one line each.
  • Exercises in class — the live activities, the prompts, and what “done” looks like for each.
  • What you’ll be able to do — the concrete capabilities you walk out with.
  • Homework — the assignment that gates the next session, with a realistic time budget.

Out of scope

These handouts are student-facing only — no facilitator hand-off cues, no recovery scripts, no speaker notes. If you are teaching a session, use the matching Facilitator Pack. The full slide decks live under Decks, and the live course pages are under Courses.
