AI Platforms

GenAI.mil — Your Primary AI Assistant

URL: genai.mil

Access: CAC login on MCEN

Authorization: MARADMIN 018/26 designates GenAI.mil as the USMC enterprise generative AI platform

GenAI.mil is the DoD enterprise generative AI platform, IL5-authorized for CUI, with over 1.1 million users across all services as of February 2026. Use it for:

  • All Course 1 exercises (Red Pen Review, Workflow Mapping, Frontier Mapping)
  • Code generation during builder courses (Courses 2–4)
  • Document drafting, analysis, and summarization
  • Brainstorming and task decomposition
  • Debugging — paste error messages and context for diagnosis

Important

GenAI.mil generates code as text output. You cannot run code inside GenAI.mil. Copy the generated code into your target platform (Power Platform, a text editor, or a development environment) to execute it.

Tips for Effective Use

  • Use the file upload feature to provide context documents (SOPs, regulations, templates)
  • When generating code for Power Platform, specify “Power Apps formula language” or “Power Automate flow” in your prompt
  • For iterative refinement, keep the conversation going in the same thread — GenAI.mil maintains context within a session
  • When debugging, paste the exact error message along with what you were trying to accomplish

CamoGPT — Advanced Capabilities

URL: camogpt.ai.army.mil

Access: CAC login on MCEN

Authorization: Available to Marines per MARADMIN 018/26

CamoGPT is an Army-managed platform with approximately 75,000 users. Use it when you need:

  • API access (for building tools that call AI programmatically)
  • Tool calling capabilities
  • Shared workspaces for team collaboration
  • Dataset conversation (upload a CSV and ask questions about it)
  • SIPR/IL6 access capability
  • Capabilities not yet available on GenAI.mil

CamoGPT and GenAI.mil are complementary. Start with GenAI.mil for most tasks. Move to CamoGPT when you need advanced features like API access, tool calling, or shared workspaces.

Building Platforms

The AI platforms above help you build. The platforms below are where your tools live.

Microsoft 365 Power Platform

Access: M365 account on MCEN

Components: PowerApps (build apps), Power Automate (automate workflows), Power BI (dashboards and reports)

This is where you BUILD tools. The AI platforms above help you build — Power Platform is where the tools live.

M365 Copilot is also available in Word, Excel, and PowerPoint on MCEN for AI-assisted document creation within Office apps.

GitHub Pages

URL: github.com

Use: Host browser-based tools accessible on MCEN (like DonDocs and this site)

For tools that need to be accessible across the enterprise without ATO concerns, browser-based applications hosted on GitHub Pages run entirely client-side with zero data transmission — no server, no database, no security footprint.

Security Reminders

  • Never enter classified information into GenAI.mil or CamoGPT (both are IL5/CUI-authorized, not classified)
  • GenAI.mil and CamoGPT are authorized for CUI — but verify your specific data handling requirements with your security manager
  • Browser-based tools hosted on GitHub Pages process data locally — no information leaves the machine
  • AI-generated content for official use must always be reviewed by a qualified human per EDD SOP
  • You are responsible for the accuracy of any AI-assisted output you submit — the AI is a tool, not an authority

Preparing for What’s Next

The current MCEN AI toolkit — GenAI.mil, CamoGPT, M365 Copilot, and Power Platform — is the starting point, not the ceiling. The DoD AI landscape is expanding rapidly:

  • GenAI.mil has announced plans for API access, agentic AI development solutions, and classified network expansion
  • AI-assisted IDEs (like GitHub Copilot, Cursor, and Claude Code) are standard in commercial development but not yet available on MCEN
  • Code execution sandboxes would allow Marines to run AI-generated code directly instead of copy-pasting between platforms
  • These tools would multiply the output of EDD-trained Marines significantly

The EDD methodology is designed to be platform-resilient. The six 201 skills — Context Assembly, Quality Judgment, Task Decomposition, Iterative Refinement, Workflow Integration, and Frontier Recognition — transfer across any AI platform. As new tools come online, Marines trained in EDD will adopt them faster because they already have the judgment framework. The tools will change. The skills won’t.

When Better Tools Arrive

When better tools arrive on MCEN, update the tool references in this curriculum. The methodology doesn’t change — only the specific platform instructions in each exercise.
